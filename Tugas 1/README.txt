Program Main.java adalah suatu program yang dapat mendata penumpang yang ingin naik, turun, dan menampilkan seluruh penumpan di bus.

Cara menjalankan program tersebut dengan run file Main.java maka akan menampilkan menu sebagai berikut :
===== BUS TRANS KOETA RADJA =====
MENU
1. Naikkan Penumpang
2. Turunkan Penumpang
3. Lihat Penumpang
Pilihan : 

jika anda memilih pilihan 1 maka anda dapat mendata orang yang akan naik bus
jika anda memilih pilihan 2 maka anda dapat menurunkan penumpang yang telah naik bus
dan jika anda memilih pilihan 3 maka anda dapat melihat semua penumpang yang berada di bus
